# Threejs Pug Boilerplate

A boilerplate for threejs and pug view engine using parcel bundler.

## Table of contents

## Overview

### Install and run

- Installing depedencies:
  `npm install`
- Compile and run code on local server
  `npm start`
- Create build
  `npm run deploy:src`
